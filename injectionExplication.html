Qu'est-ce que c'est ?

Une injection SQL est une attaque qui consiste à envoyer des requêtes SQL à un serveur de base de données. Ces requêtes peuvent être modifiées par l'attaquant pour modifier le comportement de la requête ou pour exécuter des commandes arbitraires sur le serveur de base de données.
