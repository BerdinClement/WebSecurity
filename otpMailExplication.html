Qu'est-ce que c'est ?

Qu'est-ce que c'est ?

OTP mail, est un système d'authentification via mail. Donc lors d'une tentative de connexion, un mail va être envoyé automatiquement, et il enverra un code de validation qui a une validité de 5min, il faudra donc rentrer ce code pour être enfin connecté.

Dans notre cas ?

Pour essayer, il vous faudra créer un utilisateur avec une adresse e-mail valide, sinon si vous souhaitez passer à la gestion d'un profil vous pouvez vous connecter avec l'utilisateur admin@admin.fr et avec comme mot de passe admin.

Comment résoudre ce problème ?

Pour résoudre ce problème, nous avons donc utiliser PHP Mailer, c'est une bibliothèque PHP qui va donc permettre l'envoie d'un mail. Ici nous l'enverrons de l'adresse noreply@clementberdin.fr, et vous le recevez sur le mail insérer dans l'input. (Attention, pour tester le mail, il faut un user enregistré dans la base, il est possible de créer un utilisateur), faites attention le mail peut être passer dans les indésirables.

Comment crypter ?

Ici nous en avons donc profiter pour crypter les mots de passes lors de la création d'un utilisateur, à l'aide de la fonction hash() propore à PHP, ici nous avons choisi de le faire avec l'algorithme 'sha256' qui génère des hachages irréversibles et uniques

La suite ?

Après être connecté vous allez pouvoir gérer votre profile.
